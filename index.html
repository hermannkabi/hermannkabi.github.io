About me




Hi there!

My name is Hermann Käbi, my hobby is web development. I also make mobile apps with Flutter.

I have been programming since the age of six, when I made simple text-based programs with Python. Since then, my interest has leaned towards design and user experience, but believe me, programming is still a big part of my life.

In 2020, during the lockdowns, I started learning Flutter for a school project and fell in love. After that, I starting web development and learnt the basics of HTML, CSS and JavaScript.
